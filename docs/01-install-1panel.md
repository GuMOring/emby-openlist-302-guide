# 01. 安装 1Panel

1Panel 是一个 Linux 服务器运维管理面板，可以用来管理 Docker、网站、数据库和应用服务。

## 官方文档

https://1panel.cn/

## 安装前准备

推荐系统：

- Debian 11/12
- Ubuntu 20.04+
- CentOS 7/8
- AlmaLinux
- Rocky Linux

建议配置：

- CPU：1 核以上
- 内存：1GB 以上
- 硬盘：20GB 以上

## 安装步骤

请参考 1Panel 官方安装命令，或使用以下方式安装：

```bash
curl -sSL https://resource.fit2cloud.com/1panel/package/v1/quick_start.sh -o quick_start.sh
sudo bash quick_start.sh
